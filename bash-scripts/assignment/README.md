# Finding CGPA of *CSB* students using bash script

## Task

  Convert to .txt file: may be pdftotext -layout useful
  Separate results of C4B only.
  Don't edit the files directly.
  Compute sgpa.
  Show result with name and register number.

  Compute cgpa.
